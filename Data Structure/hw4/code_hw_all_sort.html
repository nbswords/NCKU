You are asked to write a program which can

  • read an integer is_p (meaing whether to print out A or not), natural number n, a lower bound integer a_min, an upper bound integer a_max, and an integer as the random seed rs
  • allocate memory for an array A of n integers (here we assume to use A[1],...,A[n], if you want to use A[0],...,A[n-1], please do whatever necessary changes by yourself)
  • using the random seed rs to generate n random integer numbers which are uniformly distributed in the range [a_min, a_max] , and store these n random integer numbers in A[1],...,A[n].
  • if is_p=1 then print out A[1],...A[n] and STOP; Otherwise do nothing and CONTINUE
  • write the following 7 sorting functions
    • IS(A,n) : do the insertion sort
    • MS(A,1,n) : do the merge sort
    • HS(A,n) : do the heap sort
    • QS(A,1,n) : do the quick sort
    • CS(A,n,k) : do the counting sort
    • RS(A,n,k) : do the radix sort
    • BS(A,n) : do the bucket sort
    these five functions all sort the same array A in nonincreasing order (i.e.從大排至小；非從小排至大!！)
  • your code should also contain the following 4 other functions
    • printout_array(A,1,n) : to printout the integer 1-D array A (from Hw2, or HW4, make necesary changes by yourself)
    • duplicate_array(A,B,1,n) : to duplicate 1-D array A to be array B, where both A & B have dimensions 1xn (from HW2, or HW4, make necesary changes by yourself)
    • is_A_desc(A,n) : to check whether A[i]>=A[j] for any i<j
      if it passes the checking, then return "1" and printout "order OK!"; otherwise, return "0" and printout "!!WRONGLY SORTED!!". (from HW4, make necesary changes by yourself)
    • mytimer() : to return a time tag. please use related files in this example file

To be more specific, your code has to

  1. read [is_p, n, a_min, a_max, rs] from screen
  2. allocate memory for 2 arrays of integers A and B, each with size=n
  3. using rs to generate n random integers uniformly distributed in the range [a_min,a_max], store them in A
  4. if is_p=1 then print out A and STOP
  5. copy A to B (i.e. duplicate A's value into B)
  6. DO INSERTION SORT
    1. execute t1=mytimer(), IS(A,n), execute t2=mytimer(), compute t_is=t2-t1
    2. execute is_A_desc(A,n), if it returns "0" then print out "!!INSERTION SORT WRONG!!" and STOP.
  7. copy B back to A so that A has its original values
  8. DO MERGE SORT
    1. execute t1=mytimer(), MS(A,1,n), execute t2=mytimer(), compute t_ms=t2-t1
    2. execute is_A_desc(A,n), if it returns "0" then print out "!!MERGE SORT WRONG!!" and STOP.
  9. copy B back to A so that A has its original values
  10. DO HEAP SORT
    1. execute t1=mytimer(), HS(A,n), execute t2=mytimer(), compute t_hs=t2-t1
    2. execute is_A_desc(A,n), if it returns "0" then print out "!!HEAP SORT WRONG!!" and STOP.
  11. copy B back to A so that A has its original values
  12. DO QUICK SORT
    1. execute t1=mytimer(), QS(A,1,n), execute t2=mytimer(), compute t_qs=t2-t1
    2. execute is_A_desc(A,n), if it returns "0" then print out "!!QUICK SORT WRONG!!" and STOP.
  13. copy B back to A so that A has its original values
  14. DO COUNTING SORT
    1. execute t1=mytimer(), CS(A,n), execute t2=mytimer(), compute t_cs=t2-t1
    2. execute is_A_desc(A,n), if it returns "0" then print out "!!COUNTING SORT WRONG!!" and STOP.
  15. DO RADIX SORT
    1. execute t1=mytimer(), RS(A,n,k), execute t2=mytimer(), compute t_rs=t2-t1
    2. execute is_A_desc(A,n), if it returns "0" then print out "!!RADIX SORT WRONG!!" and STOP.
  16. DO BUCKET SORT
    1. execute t1=mytimer(), BS(A,n,k), execute t2=mytimer(), compute t_bs=t2-t1
    2. execute is_A_desc(A,n), if it returns "0" then print out "!!BUCKET SORT WRONG!!" and STOP.
  17. printout n, a_min, a_max, rs, t_is, t_ms, t_hs, t_qs, t_cs, t_rs, and t_bs
  18. free all the memories and end the program

Thus, if your code stops in the middle somehwere showing a warning message saying that A is not sorted in nonincreasing order, that must mean your sorting algorithm is wrong. A correct code should successfully sort the same n integers without problems.
